# Welcome to my dotfiles
## This will be a walkthrough of having a clean install of Ubuntu and how I setup my machine.

```bash
sudo apt-get install \
  build-essential \
  cmake \
  fonts-font-awesome \
  git \
  i3 \
  xclip \

sudo add-apt-repository ppa:jonathonf/vim
sudo apt-update
sudo apt-install vim
```
